# EOCMEET 安裝與啟動教學

## 1. 下載

請至 GitHub Releases 下載：

```text
EOCMEET_RC13_Security_QA_Edition.zip
```

## 2. 解壓縮

請將 ZIP 完整解壓縮到資料夾，例如：

```text
C:\EOCMEET\EOCMEET_RC13_Security_QA_Edition
```

不要只在 ZIP 壓縮檔內直接點執行，必須完整解壓縮。

## 3. 啟動

進入資料夾後，執行：

```text
EOCMeet_START.bat
```

首次啟動時會詢問安全模式：

```text
1 = 內部封閉 LAN 免 PIN
2 = PIN 保護模式
3 = 嚴格模式：PIN + 停用附件上傳
```

一般建議選擇 **2 = PIN 保護模式**。

## 4. 進入會議

啟動成功後，畫面會顯示 HTTPS 會議網址。請讓同一區域網路內的使用者，以瀏覽器開啟該網址。

常見使用方式：

- 指揮所筆電啟動 EOCMEET
- 現場人員手機連到同一 Wi-Fi
- 手機瀏覽器輸入會議網址
- 輸入名稱與 PIN 後進入會議

## 5. 常見問題

### Q1：沒有網際網路可以用嗎？

可以。EOCMEET 的主要設計就是 LAN 與離線優先。只要現場有同一個區域網路或 Wi-Fi，即可使用。

### Q2：可以放外網嗎？

不建議直接裸露到公開網際網路。本系統比較適合 LAN、VPN、衛星路由器內網或受控網路。

### Q3：手機不能開相機或麥克風？

請確認使用 HTTPS 網址，並允許瀏覽器使用相機與麥克風。

### Q4：其他人連不進來？

請確認：

- 是否在同一個 Wi-Fi 或 LAN
- Windows 防火牆是否允許連線
- 啟動畫面顯示的 IP 是否正確
- 手機是否使用行動網路而不是現場 Wi-Fi

### Q5：正式使用前要做什麼？

建議先做：

- 手機進入測試
- 電腦進入測試
- 麥克風測試
- 攝影機測試
- 螢幕分享測試
- 斷線重連測試
- 多人同時進出測試
- 管理頁與診斷頁測試
