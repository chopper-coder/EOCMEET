# EOCMEET

**EOCMEET** 是一套離線優先、區域網路優先的應急視訊會議系統，適合災害現場、指揮所、消防分隊、臨時救災基地台、無網際網路或外部雲端服務不可用時使用。

本系統的設計目標不是取代 Google Meet、Zoom 或 Teams，而是在外部網路中斷、雲端服務無法使用、災害現場需要快速建立內部通訊時，提供一套可在區域網路內運作的視訊會議、語音通訊、訊息、檔案交換與人員動態工具。

## 主要特色

- 離線優先：以 LAN、臨時 Wi-Fi、行動路由器或衛星路由器內網為主要使用場景。
- 區域網路會議：同一網段內人員可透過瀏覽器加入會議。
- 視訊與語音：支援攝影機、麥克風與多人會議。
- 螢幕分享：適合指揮所展示地圖、表單、現場影像或任務資訊。
- 人員頁與動態頁：顯示線上人員、加入與離開紀錄。
- 訊息與檔案交換：可在內網中傳遞會議訊息與必要附件。
- 管理頁與診斷頁：協助檢查服務狀態、網路狀態與版本資訊。
- 安全強化：RC13 已加入管理頁入口保護、Cookie 強化、發布包敏感檔清理與訊息 identity 修正。

## 適用情境

- 災害現場臨時指揮站
- 消防分隊與指揮中心內網會議
- 網際網路中斷時的備援會議
- 行動路由器、衛星路由器、微波通訊或臨時 Wi-Fi 內部使用
- 救災演練、教育訓練、跨單位協調測試

## 不建議情境

- 直接架在公開網際網路當雲端會議服務
- 未設定 PIN、管理密碼與防火牆就對外開放
- 取代正式商用雲端會議服務

## 下載

請到 GitHub Releases 下載最新版 ZIP：

```text
EOCMEET_RC13_Security_QA_Edition.zip
```

## 快速啟動

1. 下載 ZIP。
2. 解壓縮整個資料夾。
3. 執行 `EOCMeet_START.bat`。
4. 首次啟動依需求選擇安全模式：
   - 1 = 內部封閉 LAN 免 PIN
   - 2 = PIN 保護模式，建議一般測試使用
   - 3 = 嚴格模式，PIN + 停用附件上傳
5. 依畫面顯示的 HTTPS 網址進入會議。

## 安全提醒

- 本系統建議部署於 LAN、VPN、專線、衛星路由器內網或其他受控環境。
- 不建議直接暴露於公開網際網路。
- 正式使用前請設定會議 PIN、管理者密碼與防火牆規則。
- 請勿將 runtime secret、log、db、附件或現場資料上傳至公開 repository。
